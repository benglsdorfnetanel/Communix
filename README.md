## Communix

to run the project:

open CMD and copy:

> docker run -p 6379:6379 --name some-redis -d redis

after docker finished, open the project and run:

>npm i

and after the install:

>npm start

i used Postman and open event listeners to:

 > blast
 
 > wild
 
 > spin

the json for events:

>blast: {"userInt": 4} //4 is the number of users

>wild: {}

>spin: {}
